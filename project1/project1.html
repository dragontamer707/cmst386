How Search Engines Work

A search engine works through three primary functions: crawling, indexing and ranking. Crawling is a discovery process. The search engine sends out crawlers or spiders to find new and updated content. Indexing is where the engine stores and processes information found in an index of all the content they deem good enough to give as search results. And Ranking is for when somebody performs a search. The search engine scours the index for a highly relevant result and then orders the content. The ordering is sorted by relevance known as ranking. The higher a website is ranked, the more relevant the search engine believes it is to the search query.

Importance of Keywords and Descriptions as Meta Tags

There are three important parts of a meta tag. Title, Description and Keywords. The reason meta tags are important is that search engines read them in order to see if keywords and description are related to visible content. A meta tag is an element that describes and summarizes the content of a page for the benefit of search engines and users.

Four Elements for Writing Relevant Content Links

Assign alt text for images in HTML to give search engines a text description of the visual content, supplement search boxes with navigation and crawlable links, supplement flash or java plug-ins with text on the page and provide a transscript for video and audio content if the words and phrases used are meant to be indexed by search engines.
